What it is A Turkish football analytics project modelling the Süper Lig season using historical match data and Monte Carlo simulation to project final standings.
How it works Match data from football-data.co.uk → DuckDB + dbt transformations → logistic regression on rolling 5-match form → 10,000 Monte Carlo simulations over remaining fixtures.
Stack Python · dbt · DuckDB · MotherDuck · API-Football
